## Readme for Final Project

### Libraries

This code was developed using the Bootstrap 5 framework. Some custom CSS styling was also applied.

### How to run

Open the project's index.html using Live Server or a similar tool. The site should load. As a one
page site, index.html, styles.css, and form.js should be adequate for running the code. If any eement
of the site desn't load, check that all images are in the images folder and that the three
aforementioned files are present in the parent directory.
